# Hello Scala Akka HTTP

This is a simple HTTP server project built with **Scala 3** and **Akka HTTP**.

## Overview

The project demonstrates a basic setup for an Akka HTTP server, including:
-   Scala 3 configuration
-   Akka Actor Typed and Akka HTTP dependencies
-   A simple "Hello World" endpoint

## Documentation

-   [Get Started Guide](docs/get-started.md): Instructions on how to run and test the server.

## Quick Run

```bash
sbt run
```

The server will start at `http://localhost:8081`.
